# graphql-simpledemo
A very simple demo to go with the presentation

# Presentation URL
https://docs.google.com/presentation/d/1riZLFfKu6SJRWCU5cTFlNKEvYAfRtbB7T79nIEbk0IE/

# Install
1. `git clone https://github.com/peorth80/graphql-simpledemo.git`

## Server
1. `cd server`
2. `npm install`
3. `npm start`

## Client
1. `cd client`
2. `npm install`
3. `npm start`
